Career Partner is an AI-powered web application designed to help students and learners get personalized career guidance based on their interests, skills, and goals. The platform provides AI-generated career recommendations through an interactive and user-friendly interface.

#Features
AI-based career recommendations
Responsive and clean user interface
User-friendly career guidance flow
Real-time AI response generation using OpenRouter API
Flask backend integration
SQL database support
Deployed and accessible online

#Project Overview
The main goal of this project is to assist users who are confused about their career path by providing personalized recommendations using AI.
Users can enter their interests, skills, or career-related queries, and the system processes the input to generate relevant suggestions and guidance.
The project focuses on:
-smooth user experience
-responsive design
-AI integration
-backend routing and database handling

The project is deployed on Render.
